CST 251 Resources

Most course resources are available in different formats so you can customize this list to meet your learning style.

Required

The internet

Let's do a refresher on how the internet, and ultimately the Web, works

  • 20 Things I Learned about Browsers & the Web - A beautifully put together online book that covers the web and other internet-y things. Also note all those fancy page turning animations are pure HTML5 goodness to boot.

Introduction to Git & Version Control

  • Pro Git - You only need to read Chapters 1 & 2.
  • Pro Git Videos - View all four videos.
  • Git challenges - Become a Git rockstar with interactive terminal training from GitHub. You only need to get half-way through this training.

Suggested

I'm going to be awesome

Have some extra time to watch the inventor of the internet talk about the web? Want to peek into a browser's guts? Check out some of these resources to add some zest to your life.

Browser internals

  • How browsers work - An exhaustive overview of WebKit browser internals. Some sections are very technical, but the first few chapters are informative.
  • Faster HTML and CSS: Layout Engine Internals for Web Developers - David Baron from Mozilla talking about browser internals.

cHTeMeLe - HTML5 board game!

The world's first HTML5-certified board game can be printed and played for free.
