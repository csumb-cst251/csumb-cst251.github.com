CST 251 Web tools

Schedule

Watch out! The schedule might change during the course. Subscribe to the announcements to be notified of any corrections.

week 1

Introduction: What's the web?

We'll cover the sundry administrative requirements of the course, including the syllabus and expectations. We then launch into the history of the web, how the web actually works, and set everyone up with the necessary tools to start the course.

Inspiration for the week:
Lecture
  • Introduction to the course.
  • Discovering the industry of web development and design.
  • How the web works and browser internals.
Lab
  • Introduction to Git.
  • Installing and setting up Git.
  • Forking course repository, making commits.

week 2

HTML: The basics

In the beginning, Sir Tim Berners-Lee crated HTML. And it was good. Today we use lots of fancy technologies to power the web, but in the end we're just creating HTML markup for browsers to display. This week will cover the basics of HTML and DOM theory.

Inspiration for the week: BrowserQuest
Lecture
  • XML, HTML, and structured markup.
  • The family tree: ancestors, children, siblings.
  • Available tags and common attributes.
Lab
  • Building a web page.
  • Using the browser's HTML inspector.

week 3

HTML: HTML5 and the future

I have seen the future, and it is the browser.

Inspiration for the week:
Lecture
  • Advanced tags.
  • Structured markup, and why it is important.
  • Introduction to accessibility (a11y).
Lab
  • Building a web page.
  • Using the browser's HTML inspector.
