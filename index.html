week 5

CSS Selectors & specificity

CSS selectors can be complicates, we'll look at selector theory and the all-important rules of specificity. This is the "Cascading" part of CSS.

week 6

February 25

CSS Styles: the basics

Now that we can select different elements on the page, we should apply some style. We'll cover the basics of CSS styles, including colors, positioning, and backgrounds.

week 7

March 4

CSS Styles: advanced

OK, you can make something colorful, change it's background, and stick it anywhere you want on a screen... but can you round the corners, spin it 15 degrees, and make it play well with other elements on the page?

week 8

March 11

CSS3 styles & transitions

Style doesn't need to stand still, we can use CSS transitions to move, shake, and shimmy our way across the page. This week will cover basic CSS animations using transition states and modern browsers.

week 10

March 25

Common layout patterns in CSS

CSS is amazingly expressive for such a tiny language. We'll learn how to tame the beast using best practices and layout patterns that you can reuse over and over again.

week 11

April 1

Bootstrap, CSS frameworks, & how to write LESS css

Layout patterns are nice, but letting other people do the hard work for us is more awesomer! We'll plug into the world of CSS frameworks and CSS super-structures like SASS and LESS.

week 12

April 8

Introduction to the CMS & Drupal

The era of the "webmaster," where some dude in a trailer updated your website for $50 a month is dead. Most sites use some sort of CMS (Content Management System) to... um... manage their content. We'll get an introduction to CMSes and start working with Drupal.

week 13

April 15

Basic site building in Drupal

Even if you are never, ever, going to write a line of code in your life, understanding the CMS, and a popular one like Drupal, will give you a leg up in the world. We'll cover how to install, do basic maintenance, and play around with Drupal.
